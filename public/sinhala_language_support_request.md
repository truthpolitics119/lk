# Language Support Request: Sinhala (සිංහල)

## Overview

This document provides all translated string pairs required to add **Sinhala (si)** as a supported language on the Sri Lanka Political Accountability Record website. The site is currently available in English; Sinhala translations for all UI text, section headings, labels, and body content are provided below.

---

## Language Metadata

| Field | Value |
|---|---|
| Language Name | Sinhala |
| Native Name | සිංහල |
| Language Code | `si` |
| Script | Sinhala |
| Direction | LTR |
| Region | Sri Lanka |

---

## Implementation Notes

- **Font**: Use a web font that supports the Sinhala Unicode block (U+0D80–U+0DFF). [Noto Sans Sinhala](https://fonts.google.com/noto/specimen/Noto+Sans+Sinhala) is recommended and available via Google Fonts.
- **Language toggle**: Add `si` as a locale option alongside `en`. The language switcher label should read **සිංහල**.
- **HTML lang attribute**: Set `<html lang="si">` when Sinhala is active.
- **Line height**: Sinhala script typically requires slightly more line-height than Latin; `1.7–1.8` is recommended for body text.
- **No RTL changes needed**: Sinhala is a left-to-right script.
- **Encoding**: All Sinhala text is standard UTF-8.
